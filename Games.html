How to Play: Solar System Builder

  1. Click on a planet type (Rocky, Gas Giant, or Ice Giant) to select it
  2. Adjust the distance from the star using the slider (measured in Astronomical Units)
  3. Adjust the planet mass using the slider (relative to Earth's mass)
  4. Click "Add Planet" to place it in your solar system
  5. Click "Start Simulation" to see how your planets orbit the star
  6. Try to create a stable system where planets don't collide or get ejected!

How to Play: Atom Explorer

  1. Add protons to the nucleus to change the element (determined by atomic number)
  2. Add neutrons to create different isotopes of the same element
  3. Add electrons to complete the electron shells (affects chemical properties)
  4. Use the element selector to quickly jump to specific elements
  5. Click "Show Chemical Bonds" to see how atoms would bond with others
  6. Try to recreate common elements and see their properties!

How to Play: DNA Puzzle

  1. Examine the DNA strand with missing base pairs
  2. Select a base (A, T, C, or G) from the options
  3. Click on an empty position to place your selected base
  4. Remember the base pairing rules: A pairs with T, C pairs with G
  5. Complete the entire strand correctly to solve the puzzle
  6. Click "Replicate DNA" to see how the strand would copy itself
